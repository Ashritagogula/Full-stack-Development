HTML Block Elements

Block elements are elements that start on a new line and take up the full width available (stretches out to the left and right as far as it can).

Block Element

This is a block element.

Inline Elements

Inline elements do not start on a new line and only take up as much width as necessary.

Inline elements can be nested inside block elements.
